De Toepassingenlaag

De toepassingenlaag bestaat uit verschillende soorten software, dit zijn vooral toepassingen of applicaties. Vaak zijn dit de apps die je hebt gedownloadt op je smartphone.

Application Programming Interfaces

API's zijn de interfaces tussen de logische laag en de toepassingenlaag, een API is een software interface. API's zorgen ervoor dat er een consistentie zit in de manier van het gebruiken van een apparaat. Denk hierbij aan grafische elementen, maar ook aan verbinding maken met het internet.
